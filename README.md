# Unit & E2E Test workshop

This repository contains a small e-commerce project for our workshop about unit & e2e tests

### Run project

```bash
# create .env.local and put REACT_APP_API_URL
REACT_APP_API_URL=http://localhost:3001

# run react app
npm run start

# run http server
npm run server

# run cypress
npm run cypress

# run tests
npm run test
```
